## Today Core Implementation

오늘의 핵심 구현은 발표 시나리오가 아니라 실제 챗봇 기능의 데이터 흐름이다.

1. 청년지원사업 챗봇의 기본 프로필 슬롯을 정리한다.
2. 고용24 훈련과정 API를 실제 연결 가능한 핵심 API로 둔다.
3. 고용24 채용정보 API는 개인키 제한을 감지하고, 직접 채용공고 조회가 막히면 채용 탐색 가이드로 fallback한다.
4. 온통청년은 키 발급 전까지 스키마/normalizer 설계를 유지하되, 대체 정책 데이터로 보완하지 않는다.
5. 기업마당은 오늘 핵심 구현에서 제외하고, 창업/사업자 질문 보조 데이터로만 남긴다.
6. Agent가 API 결과가 없거나 조건이 부족한 상태에서 정책/훈련/채용 정보를 지어내지 않도록 한다.
